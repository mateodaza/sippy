## O que acontece

Se você tentar enviar dinheiro para um número de telefone que ainda não está no Sippy, **a
transferência não é concluída.** Seus fundos permanecem na sua carteira.

Em vez disso, o Sippy envia ao destinatário um convite pelo WhatsApp para que ele possa se cadastrar.

## O fluxo

1. **Você envia normalmente.** Mande uma mensagem ao Sippy no WhatsApp ou use a carteira web. Digite
   o número de telefone do destinatário e o valor.

2. **O Sippy verifica se a pessoa tem uma carteira.** Se não tiver, o envio é pausado.

3. **A pessoa recebe um convite pelo WhatsApp.** O destinatário recebe uma mensagem do Sippy
   convidando-o a criar uma carteira. A mensagem inclui o valor que você tentou
   enviar.

4. **Seu dinheiro continua com você.** Nenhum fundo sai da sua carteira. Você mantém seu saldo.

5. **Assim que a pessoa se cadastrar, você pode enviar de novo.** Depois que o destinatário criar a
   carteira dele, basta enviar o dinheiro novamente.

## Como é o convite?

O destinatário recebe uma mensagem do Sippy pelo WhatsApp, algo como:

> "Alguém quer te enviar $10 no Sippy. Toque aqui para configurar sua carteira."

## Bom saber

- **Seus fundos nunca correm risco.** Nenhum dinheiro se movimenta até que tanto o remetente quanto o destinatário
  tenham carteiras.
- **O convite conta para um limite diário** -- você pode enviar alguns convites por dia
  para evitar spam.
- **Você pode verificar se alguém está no Sippy** tentando enviar. Se a pessoa não estiver, você vai
  saber na hora.
